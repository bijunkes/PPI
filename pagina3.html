CSS

Linguagem usada para estilizar e formatar a aparência de páginas web. Enquanto o HTML define a estrutura e o conteúdo de um site, o CSS controla como esse conteúdo é apresentado visualmente, incluindo coisas como cores, fontes, layouts e espaçamento.

  1. CSS 1 (1996)
  2. CSS 2 (1998)
  3. CSS 3 (2008)

Podemos incluir código CSS de 3 formar diferentes

CSS externo
Quando definimos as regras CSS a serem utilizadas por meio de um arquivo externo .css, vinculando ele ao html por meio da tag <link>
CSS incorporado no documento
Quando definimos as regras CSS por meio da tag <style>
CSS inline
Quando definimos código css de estilização no próprio elemento HTML: Texto de cor vermelha com CSS inline
