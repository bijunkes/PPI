O que é JavaScript?

O JavaScript é uma linguagem de programação criada em 1995 por Brendan Eich com o objetivo de permitir interatividade nas páginas da web, como validação de formulários e manipulação de elementos da interface diretamente no navegador, algo que HTML e CSS não podiam fazer sozinhos.

Inicialmente executado apenas no navegador, o JavaScript hoje também pode ser executado no servidor, por exemplo, com o Node.js, permitindo o desenvolvimento completo de aplicações web com uma única linguagem.

No navegador, o JavaScript é executado dentro de uma sandbox, um ambiente isolado que limita o acesso direto ao sistema do usuário por questões de segurança.

Dentro desse ambiente, o JavaScript tem acesso a objetos globais como document, que representa a estrutura HTML da página; console, usado para registrar mensagens de depuração; e alert, que exibe caixas de diálogo para interações simples com o usuário.
