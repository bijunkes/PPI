Listas com comando básicos do git

  • git init - Cria um repositório em uma pasta vazia
  • git add - Adiciona modificações do wk no stage
  • git status - Mostra o estado atual do repositório, modificações dos arquivos versionados
  • git commit - Cria um commit no repositório local
  • git push - Envia modificações do repositório local para o remoto
  • git log - Exibe no terminal os commits do repositório na branch atual
  • git log --online --all - Exibe todos os commits de forma resumida
  • git branch - Lista as brances do repositório
  • git checkout - Muda de branch
  • git checkout -b - Cria e moda para uma nova branch
